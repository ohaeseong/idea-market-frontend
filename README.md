## 브레인 쇼핑몰은 상품을 거래하는 쇼핑몰이 아닌 아이디어를 상품으로 사고 파는 쇼핑몰입니다.

개발 기간:  약 일주일

개발 인원: 1명

개발 분야: 백엔드, 프론트 엔드 (Node.js, React.js)

# 개발 기술 스택

1. **Node.js (Express)**
2. **Mysql**
3. **Sequelize**
4. **React.js**
5. **Mobx**

# 기능 명세

- **주요** **제품 관련  기능**
    1. 제품 등록
    2. 제품 조회
    3. 제품 장바구니 추가/삭제
    4. 제품 상세 조회 
    5. 장바구니 목록 조회
- **사용자 관련 기능**
    1. 로그인
    2. 내 정보 조회
